# SIWebView

封装UIWebView和WKWebView相关SDK,并提供相关注册JS的接口

## 安装
#### 1. 添加引用
依赖于WebViewJavascriptBridge
#### 2. 导入文件
下载项目后,将项目下`Classes`目录添加到项目中

## 使用
1. 导入头文件

    ```objective-c
	#import "SIWebView.h"
	```
2. 使用

	详细使用方法请看API接口
